Claude Code連携によるAI生成ブログ記事の章別分割インポート機能を提供するWordPressプラグインです。

== 説明 ==

ブログジェネレーターは、AI生成された高品質なブログ記事をWordPressのブロックエディタに完全対応した形式でインポートできるプラグインです。

主な機能:
* 章別分割インポート処理（リード文→各章→まとめ文の順次追加）
* ブロックエディタ完全対応の自動変換（段落・見出し・表・リスト・装飾）
* 画像自動アップロード・配置（アイキャッチ・章別サムネイル）
* メタディスクリプション・SEO設定の自動適用
* 日本語コンテンツ完全サポート
* 詳細デバッグログによるトラブルシューティング

プラグインはoutputsフォルダから記事ファイルを読み込み、以下の構成で完全なWordPress投稿を作成します:
- リード文（読者の関心を引く導入部）
- 章見出し＋サムネイル画像
- 高品質な本文コンテンツ（表・リスト・テキスト装飾対応）
- まとめ文（行動促進・記憶定着最適化）
- アイキャッチ画像（SNSシェア対応）

== インストール方法 ==

1. `blog-generator` フォルダを `/wp-content/plugins/` ディレクトリにアップロード
2. WordPress管理画面の「プラグイン」メニューでプラグインを有効化
3. 管理メニューの「ブログジェネレーター」にアクセス
4. アウトラインファイルを選択してインポート処理を開始

== 使用方法 ==

1. 記事ファイルがプラグインディレクトリ内の `outputs/` フォルダに配置されていることを確認
2. 管理画面 > ブログジェネレーター に移動
3. ドロップダウンからアウトラインファイルを選択
4. 「章を読み込み」をクリックして記事構造をプレビュー
5. 「インポート開始」をクリックしてインポート処理を実行
6. 進行状況とデバッグ情報を監視
7. インポート完了後、必要に応じて投稿を編集

== ファイル構成 ==

プラグインはoutputsフォルダ内の以下のファイル構成を想定しています:
- `*_outline_*.md` - 記事アウトラインファイル（タイトル・メタディスクリプション）
- `*_article_*_chapter*.md` - 章別コンテンツファイル（高品質本文）
- `*_lead_*.md` - リード文ファイル（読者引き込み最適化）
- `*_summary_*.md` - まとめ文ファイル（行動促進・記憶定着）
- `*.png` - 画像ファイル（アイキャッチ・章別サムネイル）

== よくある質問 ==

= サポートされているファイル形式は？ =
テキストコンテンツはMarkdownファイル（.md）、画像はPNGファイルをサポートしています。

= 複数記事の同時インポートは可能？ =
安定性と適切なエラーハンドリングを確保するため、現在は1記事ずつのインポートに対応しています。

= インポートが失敗した場合は？ =
詳細なデバッグログ機能により、問題の特定と解決をサポートします。

= Claude Codeとの連携方法は？ =
Claude Codeで「ブログ投稿」コマンドを実行すると、自動的に記事データが生成され、このプラグインでWordPressにインポートできます。

= 生成される記事の品質は？ =
ファクトチェック済み・引用元追加済みの高品質コンテンツで、出版レベルの信頼性を確保しています。
